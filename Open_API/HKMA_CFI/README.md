# HKMA 網絡防衛計劃（CFI）資料庫

## 資料夾說明
本資料夾存放香港金融管理局（HKMA）**網絡防衛計劃（Cybersecurity Fortification Initiative, CFI）**相關官方文件。

## 三大支柱
| 支柱 | 名稱 | 描述 |
|------|------|------|
| 1 | 網絡防衛評估框架（C-RAF） | 評估銀行網絡風險同成熟度 |
| 2 | 專業培訓計劃（PDP） | 培訓網絡安全專業人員 |
| 3 | 網絡風險資訊共享平台（CRISP/CISP） | 銀行間共享網絡威脅情報 |
