# HKMA Supervision Guideline for Licensed Stablecoin Issuers

> 來源：[Guideline on Supervision of Licensed Stablecoin Issuers](https://www.hkma.gov.hk/media/eng/doc/key-functions/ifc/stablecoin-issuers/Guideline_on_supervision_of_licensed_stablecoin_issuers_eng.pdf)
> 發布日期：August 2025
> 法律基礎：Stablecoins Ordinance (Cap. 656) s.171(4)

---

## 1. 基本定位

| 項目 | 內容 |
|------|------|
| 適用對象 | 持牌穩定幣發行人 (Licensee) |
| 發布機構 | HKMA |
| 性質 | 「must」= 法定要求；「should」= 監管期望 |
| 牌照持續要求 | 持牌後須持續符合Schedule 2最低標準 |

## 2. 儲備資產管理 (Reserve Assets Management)

### 2.1 核心要求：全額支持 (Full Backing)

| 要求 | 細節 |
|------|------|
| **基本原則** | 儲備資產市值 ≥ 未贖回穩定幣面值 |
| **額外緩衝** | 須有適當超額抵押 (over-collateralisation) |
| **估值方法** | 以bid price計算（較審慎） |
| **估值來源** | 須合理、可靠，反映市場價格 |

**特殊情况：**
- 被凍結的穩定幣（如執法行動或法院命令）也須全額支持
- 帳戶費用不得從儲備資產帳戶扣除（否則影響全額支持）

### 2.2 合格儲備資產 (Eligible Assets)

| 類型 | 具體要求 |
|------|----------|
| (i) 現金 | - |
| (ii) 銀行存款 | 期限不超過3個月 |
| (iii) 可銷售債務證券 | 須為政府/央行/主權實體/合格國際組織/多邊開發銀行發行或擔保；剩餘期限≤1年；0%風險權重（標準信用風險方法）；高流動性；不得為金融機構或關聯實體之債務 |
| (iv) 隔夜回購協議（cash receivable） | 最低交易對手風險，以(iii)資產作抵押 |
| (v) 投資基金 | 專為管理持牌人儲備資產而設立 |
| (vi) 其他HKMA可接受之資產 | - |

> **注意：** 代幣化形式之合格資產也可接受，但須向HKMA證明其符合高質量、高流動性、最低投資風險要求

### 2.3 儲備資產隔離及保管

| 要求 | 說明 |
|------|------|
| **隔離原則** | 各類穩定幣之儲備資產須與其他儲備資產池分開 |
| **债权人保護** | 須充分保障儲備資產免受其他債權人索償 |
| **自有資產分開** | 須與持牌人自有資產分開 |
| **信任安排** | 須建立有效信任安排（獨立受託人或信任宣言） |
| **事先批准** | 實施信任安排前須取得獨立法律意見並提交HKMA |
| **保管人資格** | 持牌銀行或其他HKMA可接受之資產保管人 |

**第三方資產管理：**
- 投資經理人須具備相關資質
- 持牌人仍須為儲備資產管理承擔主要責任

### 2.4 儲備資產申報及披露

| 頻率 | 要求 |
|------|------|
| **每日** | 編製未贖回穩定幣面值及儲備資產市值/組合報表 |
| **每週** | 向HKMA報告；並在網站顯眼位置更新 |
| **季度** | 檢討儲備資產管理政策及程序 |
| **按HKMA要求** | 提交每日報表 |

**獨立核證：**
- 須聘請符合HKMA要求之獨立外部核數師
- 核證內容：市值/儲備資產組合、未贖回面值、是否全額支持
- 須涵蓋期內最後一個營業日及至少一個隨機選擇之營業日
- 核證報告須於期後1個月內提交HKMA並公開

**即時報告觸發條件：**
- 違反儲備資產管理法規
- 儲備資產管理政策重大不合規
- 對帳中存在未解決差異

## 3. 發行、贖回及分銷 (Issuance, Redemption and Distribution)

### 3.1 發行要求

| 要求 | 說明 |
|------|------|
| **發行對象** | 僅可向客戶發行 |
| **發行時機** | 收到款項及有效發行請求後，須在切實可行的範圍內盡快發行 |
| **款項貨幣** | 須與掛鉤貨幣一致 |
| **鑄幣匹配** | 鑄造穩定幣須與相應增加的儲備資產池匹配 |

### 3.2 贖回要求

**持牌人須提供以下權利：**

| 權利 | 內容 |
|------|------|
| **面值贖回權** | 持有人有權按面值贖回 |
| **指定儲備資產池處置權** | 持有人有權指示處置儲備資產池（按比例）以贖回 |
| **不足額索償權** | 若儲備資產池處置所得不足，持有人有權向持牌人索償差額 |
| **破產時權利** | 以上權利在持牌人破產時仍可行使 |

**贖回機制要求：**

| 要求 | 說明 |
|------|------|
| **處理時限** | 收到有效贖回請求後**1個營業日內**處理 |
| **費用限制** | 不得收取不合理費用 |
| **條件限制** | 不得附加過度繁重條件 |
| **款項來源** | 從儲備資產池扣除，對應減少未贖回面值 |

**須取得獨立法律意見**證明已向持有人提供上述權利

### 3.3 分銷要求

| 要求 | 說明 |
|------|------|
| **第三方分銷商盡職審查** | 須考慮規模、能力、專業知識、往績、聲譽 |
| **分銷商資質** | 在香港分銷之第三方須為允許之要約人 |
| **二級市場流動性** | 須考慮是否需要及範圍 |
| **潛在利益衝突** | 須識別及適當管理/緩解 |
| **合規責任** | 須確保分銷安排符合相關司法管轄區法規 |

### 3.4 客戶入職及管理

| 要求 | 說明 |
|------|------|
| **AML/CFT盡職審查** | 須按AML/CFT指引進行客戶盡職審查 |
| **司法管轄區合規** | 須遵守經營業務之相關法規 |
| **地域限制核查** | 須核查身份證明文件、IP位置等，確保不在禁止司法管轄區運營 |
| **VPN檢測** | 須實施控制措施以緩解位置欺騙風險 |

## 4. 業務活動 (Business Activities)

### 4.1 業務活動限制

| 要求 | 說明 |
|------|------|
| **原則** | 須事先取得金融管理專員書面同意，方可從事牌照穩定幣業務以外之業務 |
| **額外要求** | 須建立清晰管治安排；進行風險評估；實施控制措施 |
| **衝突管理** | 須確保其他業務不會對牌照業務造成重大風險 |
| **資源要求** | 須確保有充足資源投入牌照業務 |

> **例外：** 認可機構（銀行）之持牌人須遵守《銀行業條例》

### 4.2 多類型穩定幣發行

| 要求 | 說明 |
|------|------|
| **原則** | 須先與HKMA討論（如欲發行額外類型穩定幣） |
| **評估要求** | 須證明具備管理多類型之能力及資源 |
| **風險評估** | 須評估對現有發行活動之影響 |

### 4.3 發行穩健性要求

持牌人須：
- 採用與穩定幣穩健可靠原則一致的策略
- 備有可行、有切實依據並可持續之業務計劃
- 以審慎方式經營業務
- 遵守相關司法管轄區之法規
- 具備執行業務計劃之能力及資源

## 6. 風險管理 (Risk Management)

### 6.1 風險管治

| 要求 | 說明 |
|------|------|
| **高級管理層責任** | 須對風險管理承擔最終責任 |
| **獨立風險管理職能** | 須建立獨立風險管理職能 |
| **風險文化** | 須建立與風險偏好一致之風險文化 |

### 6.2 風險管理框架

| 組成部分 | 內容 |
|----------|------|
| 信用風險管理 | 識別、計量、控制及監察信用風險 |
| 流動性風險管理 | 確保能履行贖回義務 |
| 市場風險管理 | 管理匯率、利率等市場風險 |
| 操作風險管理 | 系統、流程、人為錯誤等 |
| **科技風險管理** | 見6.5節 |
| 名譽風險管理 | 見6.7節 |

### 6.3 科技風險管理（節錄）

> 這是穩定幣特有的重要風險領域

| 風險類別 | 要求 |
|----------|------|
| **智能合約風險** | 須進行安全審計、獨立驗證 |
| **區塊鏈底層技術** | 須評估網絡穩定性、安全性 |
| **私鑰管理** | 須有安全密钥管理機制 |
| **系統災難** | 須有業務連續性計劃 |
| **網絡安全** | 須實施網絡安全控制 |

### 6.4 事故管理、業務連續性及退出

| 要求 | 說明 |
|------|------|
| **事故管理** | 須建立事故應對機制 |
| **業務連續性** | 須確保關鍵業務可持續運作 |
| **退出計劃** | 須有詳細退出計劃（見6.8.17） |

## 7. 企業管治 (Corporate Governance)

### 7.1 基本要求

| 要求 | 說明 |
|------|------|
| **董事會責任** | 須對業務運作及風險管理承擔最終責任 |
| **獨立非執行董事** | 須具備獨立非執行成分 |
| **委員會** | 須設立審計委員會、風險委員會等 |
| **薪酬** | 須與風險狀況一致 |

### 7.2 適當性 (Fitness and Propriety)

| 要求 | 說明 |
|------|------|
| **董事/管理人員** | 須符合適當人選標準 |
| **主要股東** | 須為適合持有控股權之人 |
| **持續評估** | 須持續監察是否符合標準 |

### 8.1 資訊及會計系統

## 10. 摘要對比表

| 範疇 | AML/CFT指引 | Supervision指引 |
|------|-------------|-----------------|
| **法律基礎** | SO s.171 + AMLO s.7 | SO s.171(4) |
| **重點** | 反洗錢/反恐融资 | 儲備資產、發行、贖回、操守 |
| **強制性** | 「must/should」= 法定要求 | 同上 |
| **關鍵詞** | 風險為本、RBA、STR | 全額支持、儲備資產隔離、贖回權 |

## 11. 實務重點摘要

### 持牌人必須牢記：

1. **儲備資產等於或超過未贖回面值**
2. **儲備資產必須隔離並由獨立保管人持有**
3. **每週向HKMA報告儲備資產狀況**
4. **1個營業日內處理贖回請求**
5. **須為持有人提供面值贖回權及不足額索償權**
6. **AML/CFT系統必須持續運行**
7. **不可支付利息**
8. **未經批准不得從事其他業務**
